# Snake Game en Pygame
Un juego de serpiente clásico desarrollado con Python y Pygame.

## Cómo ejecutarlo
1. Instala las dependencias: `pip install pygame`
2. Ejecuta el juego: `python main.py`

## Controles
- **Flechas**: Mover la serpiente.
- **Objetivo**: Comer la fruta roja para crecer y sumar puntos. 
- **Perder**: Chocar con las paredes o contigo mismo reinicia el juego.
